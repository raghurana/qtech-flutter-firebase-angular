Add New Appointments
Use this screen to add new appointments in bulk.
  • Each new appointment must be on a single line.
  • Line Format is: starttime-patientcode-doctorcode
  • For example, 210pm-js321-daf
  • where js, is initials for a patient John Smith
  • 321, is the last 3 digits of patient's mobile number
  • and, daf is code for Doctor Alexander Fleming
